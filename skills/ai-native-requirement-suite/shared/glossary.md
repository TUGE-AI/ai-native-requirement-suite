# Shared Glossary

status: `skeleton_v0.15_fix`

## 核心术语

| Term | Meaning |
|---|---|
| `idea` | 尚未澄清的想法、机会点或方向 |
| `demand brief` | 已澄清到可进入 PRD/spec 生成的需求 brief |
| `PRD/spec` | 人类可读需求规格 |
| `requirement asset package` | 面向 AI coding/testing 和人类评审的多文件需求资产包 |
| `Feature` | 能力级资产，承载目标、价值、范围、边界和 Story 引用 |
| `Story` | 一等执行资产，作为研发、测试、验收和估算的工作单元 |
| `Rule/GWT` | Story 内部的行为规约和验收场景，不替代 Story narrative |
| `Gate` | `ready` / `need_revision` / `blocked` 质量门禁判断 |
